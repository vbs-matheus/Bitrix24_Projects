# Bitrix24 Projects

Repositório que centraliza projetos de integração, automação e manipulação de dados no CRM Bitrix24, com foco em otimização de processos empresariais.

---

## 📌 Visão Geral

Este repositório reúne implementações práticas envolvendo:

* Integração entre sistemas via API
* Automação de fluxos dentro do CRM
* Tratamento e sincronização de dados
* Execução de rotinas automatizadas baseadas em eventos

Os projetos estão organizados por pastas, cada uma representando uma solução independente.


## 🚀 Projetos

### 🔹 bitrix-integrations

Conjunto de integrações entre o CRM Bitrix24 e outras plataformas externas, utilizando APIs e webhooks para comunicação entre sistemas.

**Principais características:**

* Conexão entre Bitrix24 e serviços externos
* Uso de webhooks para disparo de eventos
* Processamento e tratamento de dados dentro do fluxo
* Execução de funções em ambiente serverless (AWS Lambda)

---

### 🔹 bitrix-users-update

Projeto voltado para atualização e sincronização de usuários dentro do CRM Bitrix24.

**Principais funcionalidades:**

* Atualização de dados de usuários
* Inclusão de novos usuários em grupos de trabalho
* Comparação de dados entre o CRM e bases externas
* Execução baseada em scripts diretos com acesso às bases do CRM

---

## 🛠️ Tecnologias Utilizadas

### Backend & Integração

* Python
* APIs REST
* Webhooks
* AWS Lambda

### Plataforma

* Bitrix24 CRM

### Interface / Templates

* HTML (utilizado para estruturação de notificações e visualização de dados em integrações específicas)

---

## ⚙️ Como os Projetos Funcionam

Os projetos seguem dois padrões principais:

* **Execução por evento:**
  Scripts acionados via webhook ou integrações externas, reagindo a ações dentro do CRM

* **Execução direta:**
  Scripts executados manualmente para sincronização ou atualização de dados
